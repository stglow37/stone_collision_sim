# 🪨 Stone Collision Simulation (돌 충돌 시뮬레이션)
(사실 잘 모름)
이 프로젝트는 돌의 충돌을 물리적으로 계산하고 시각화하는 시뮬레이션 프로그램입니다.

## 📁 폴더 구조
- `version 1/`: 주요 실행 소스 코드가 포함된 폴더입니다.
  - `main.py`: 프로그램의 진입점(Entry Point)입니다.
  - `physics.py`: 충돌 및 물리 연산 로직이 담겨 있습니다.
  - `graphics.py`: 시뮬레이션 시각화 관련 코드입니다.
  - `config.py`: 설정 값(변수) 관리 파일입니다.
  - `model.py`: 객체 모델 정의 파일입니다.
- `test*.py`: 개별 기능 테스트를 위한 스크립트입니다.

1. Python이 설치되어 있는지 확인합니다.
